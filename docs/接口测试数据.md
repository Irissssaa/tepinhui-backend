## 时间字段格式说明

| 字段类型 | 格式 | 示例 |
|---|---|---|
| `DATE`（日期）| `yyyy-MM-dd` | `2024-04-15` |
| `DATETIME`（日期时间）| `yyyy-MM-ddTHH:mm:ss` | `2024-04-15T10:30:00` |
| 注意 | T 是固定字符，连接日期和时间部分 | —— |

## 一、溯源模块 C端 `/tph/api/v1/trace`

> 查询接口公开，录入接口需商家角色。

### 1.1 查询溯源（公开）
```
GET /api/v1/trace/{traceCode}
```
**示例：**
```
GET /api/v1/trace/TP-ZJ-LJ-2024-000001
```

### 1.2 商品溯源列表（公开）
```
GET /api/v1/trace/product/{productId}
```
**示例：**
```
GET /api/v1/trace/product/1
```

### 1.3 录入溯源（商家）
```
POST /api/v1/trace
Header: Authorization: Bearer ***
Content-Type: application/json

{
    "productId": 1,
    "batchNo": "B20240415001",
    "originAddress": "浙江省杭州市西湖区龙井路88号",
    "longitude": 120.129870,
    "latitude": 30.251830,
    "produceDate": "2024-04-05",
    "producerName": "杭州狮峰龙井茶业有限公司",
    "rawMaterial": "鲜叶（龙井43号品种）",
    "processFactory": "杭州龙井茶厂",
    "processDate": "2024-04-06",
    "processDesc": "摊放→青锅→回潮→辉锅→整理，共6道工序",
    "inspectOrg": "浙江省茶叶质量检测中心",
    "inspectDate": "2024-04-08",
    "inspectResult": "合格",
    "inspectReportUrl": "https://example.com/reports/inspection2024.pdf",
    "warehouseInTime": "2024-04-09T08:00:00",

    "logisticsInfo": "从杭州仓出发，经顺丰速运至上海仓，配送至客户"
}
```

**溯源码格式：** `TP-{省份}-{品类}-{年份}-{6位随机数}`

**省份缩写参考：** 浙江→ZJ、江苏→JS、安徽→AH、福建→FJ、广东→GD、四川→SC 等

**品类缩写参考：** 龙井→LJ、碧螺春→BLC、普洱→PE、乌龙→WL、白茶→BC 等

## 二、溯源模块 管理端 `/tph/api/v1/trace/admin`

> 需要管理员角色。

### 2.1 待审核列表
```
GET /api/v1/trace/admin/pending?page=1&size=10
Header: Authorization: Bearer ***
```

### 2.2 审核详情
```
GET /api/v1/trace/admin/{id}
Header: Authorization: Bearer ***
```

### 2.3 所有溯源列表
```
GET /api/v1/trace/admin/list?page=1&size=10
Header: Authorization: Bearer ***
```

### 2.4 审核溯源
```
PUT /api/v1/trace/admin/audit
Header: Authorization: Bearer ***
Content-Type: application/json

{
    "id": 1,
    "status": "pass",
    "remark": "信息核实无误，审核通过"
}
```

**status 可选值：** `pass`（通过）、`reject`（拒绝）

## 三、特产模块 `/tph/api/v1/specialties`

```

### 3.2 特产详情（公开）

GET /api/v1/specialties/{id}
```

```
GET /api/v1/specialties/1
```

## 四、地图模块 `/tph/api/v1/map`

### 4.1 特产分布地图（公开）
```
GET /api/v1/map/specialties
```
**返回说明：** 每条记录包含 `region`（省份）、`specialtyName`（特产名）、`count`（该省特产数量）、`longitude`、`latitude`

### 4.2 节气推荐（公开）
```
GET /api/v1/map/season-recommend
```
**返回说明：** 根据当前节气自动匹配，返回推荐特产列表，`reason` 字段标注当前节气

---

## 五、统计模块 `/tph/api/v1/stats`

### 5.1 溯源统计总览（公开）
```
GET /api/v1/stats/trace
```
**返回说明：** 返回 `totalTraceCount`（总数）、`passedTraceCount`（通过）、`pendingTraceCount`（待审）、`rejectedTraceCount`（驳回）

### 5.2 区域溯源统计（公开）
```
GET /api/v1/stats/trace/by-region
GET /api/v1/stats/trace/by-region?region=浙江
```
**返回说明：** 各省份溯源记录数，支持按省份筛选

### 5.3 区域销售统计（公开）
```
GET /api/v1/stats/sales/by-region
GET /api/v1/stats/sales/by-region?region=广东
```
**返回说明：** 各省份销售数据，支持按省份筛选
