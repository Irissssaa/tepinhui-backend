## 三、技术配置详情

### 3.1 当前 pom.xml 依赖

| 依赖 | 版本 | 用途 |
|---|---|---|
| spring-boot-starter-web | 3.5.14 | Web 开发 |
| spring-boot-starter-validation | 3.5.14 | 参数校验 |
| spring-boot-starter-security | 3.5.14 | 安全认证 |
| spring-boot-starter-data-redis | 3.5.14 | Redis 数据访问 |
| spring-boot-starter-cache | 3.5.14 | 缓存抽象层 |
| mybatis-plus-spring-boot3-starter | 3.5.5 | ORM 持久层 |
| springdoc-openapi-starter-webmvc-ui | 2.3.0 | Swagger 文档 |
| mysql-connector-j | - | MySQL 驱动 |
| jjwt | 0.12.5 | JWT 认证 |

### 3.2 application.properties 主要配置

**Redis 配置：**
```properties
spring.data.redis.host=localhost
spring.data.redis.port=6379
spring.data.redis.password=
spring.data.redis.database=0
```

**MyBatis-Plus 配置：**
```properties
mybatis-plus.mapper-locations=classpath:mapper/**/*.xml
mybatis-plus.configuration.map-underscore-to-camel-case=true
mybatis-plus.global-config.db-config.id-type=auto
mybatis-plus.global-config.db-config.logic-delete-field=deleted
```

**Swagger 配置：**
```properties
springdoc.api-docs.path=/api-docs
springdoc.swagger-ui.path=/swagger-ui.html
```
